Get Bucket List Specification

Date Created : 07/27/2017 Version : 1.0.0 Owner : Osanda Deshan Description : This is an executable specification file which follows markdown syntax. Every heading in this file denotes a scenario. Every bulleted point denotes a step.

tags: get_bucket_list

  • Given that a user invokes "Get Bucket List"

Get bucket list using valid token

  • When the user invokes the BucketList API using token <file:/resources/access_tokens/valid_token.txt>
  • Then the status code for retrieving bucket list request is "200"
  • And the response for retrieving bucket list request contains the following
    Key Value
    id[0] 1
    title[0] string
    description[0] string
    goalDate[0] 2017-08-26T05:03:21.952

Get bucket list using invalid token

  • When the user invokes the BucketList API using token <file:/resources/access_tokens/invalid_token.txt>
  • Then the status code for retrieving bucket list request is "401"

Get bucket list using no token

  • When the user invokes the BucketList API using token <file:/resources/access_tokens/null_token.txt>
  • Then the status code for retrieving bucket list request is "401"
